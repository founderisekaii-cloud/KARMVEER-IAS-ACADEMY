# **App Name**: Karmveer Academy Online

## Core Features:

- Dynamic Course Listings: Display courses with filters for exam type, batch timing, and fees.
- Online Admission System: Student registration form, admission ID generation, and automated confirmation emails/SMS.
- Payment Gateway Integration: Integrate UPI and Razorpay for online fee payments, with admin control over prices and payment modes.
- Results and Notices: Upload and display student results with photos, ranks, and exam notifications. Leverage Google Drive for file storage.
- Admin Dashboard: Secure admin login to manage courses, student data, payments, and send announcements via email/SMS.
- AI-Powered Study Plan Generation: Utilize an AI tool that generates personalized study plans based on the student's exam goals, time availability, and previous performance, integrating course details and exam schedules.
- Automated Content Generation for Exam Updates: Generate summaries and key points from exam notifications uploaded as PDFs or text, powered by AI to streamline information delivery to students.

## Style Guidelines:

- Primary color: Saffron (#FF9933), symbolizing courage and sacrifice, reflecting India's rich heritage.
- Background color: Light gray (#F0F0F0), offering a neutral backdrop that enhances readability and focuses attention on the content.
- Accent color: Blue (#3385FF), representing depth, stability, and trust, inspired by the blue in the Ashoka Chakra. 
- Body text: 'PT Sans', sans-serif, providing a clean and readable font suitable for lengthy content.
- Headline font: 'Playfair', serif, bringing elegance and traditional aesthetic to highlight headlines. Use 'PT Sans' as body font with 'Playfair' header font. 
- Use icons that represent academic achievements, Indian culture, and government service themes.
- Subtle animations for loading screens, form submissions, and transition effects to improve user engagement.
